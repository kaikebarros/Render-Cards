React Render Cards

Projeto desenvolvido para praticar os fundamentos do React.

Tecnologias utilizadas

- React
- Vite
- JavaScript
- CSS

Objetivo

Este projeto foi criado para exercitar conceitos essenciais do React, como:

- Componentes reutilizáveis
- Props
- Renderização de listas com "map()"
- Organização de arquivos e componentes

Funcionalidades

- Exibição de cards de pessoas.
- Cada card recebe informações por meio de Props.
- Interface simples e responsiva.

Como executar

1. Clone este repositório.
2. Instale as dependências:

npm install

3. Inicie o projeto:

npm run dev

Aprendizados

Durante o desenvolvimento deste projeto, pratiquei a criação de componentes reutilizáveis, a passagem de dados com Props e a renderização dinâmica de elementos utilizando React.
